# theNews
This project collects the latest news headlines from the Brazilian news portal G1 and performs a simple sentiment analysis based on keyword matching. It classifies news as positive or negative and displays the results in a bar chart.

### Features
* Scrapes real-time news headlines from G1
* Performs basic sentiment analysis using keywords
* Displays results in a bar chart using Matplotlib
* Prints all collected headlines with the current date

### Technologies Used
* Python
* Pandas
* Requests
* BeautifulSoup
* Matplotlib
